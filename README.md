# BaiThucHanh_JSP

Bài thực hành JSP gồm các nội dung:

- Bài 1: Hiển thị bảng nhân 2 bằng JSP
- Bài 2: Hiển thị bảng nhân 2 với dấu ? cho kết quả chưa tính
- Bài 3: Hiển thị bảng cửu chương từ 2 đến 9
- Bài 4: Tính giai thừa từ 0 đến 10
- Bài 5: Nhập tên từ form HTML và hiển thị lời chào bằng JSP

## Công nghệ sử dụng
- Java JSP
- Apache Tomcat
- NetBeans IDE

## Cách chạy
- Mở project bằng NetBeans
- Run Project
- Truy cập các file `.jsp` trong thư mục Web Pages
